## Getting started
1. Es recomendable crear un entorno virtual e instalar las dependencias:

    Bash:
    ```bash
    python -m venv env
    source env/Scripts/activate
    pip install -r requirements.txt
    ```
    Windows cmd:
    ```cmd
    python -m venv env
    env/Scripts/activate
    pip install -r requirements.txt
    ```

2. Crear el archivo .env:
